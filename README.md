# Guess CLI

A whimsical command-line number guessing game, wherein the Bard's own spirit doth animate the terminal!
The computer hath pondered a secret numeral 'twixt 1 and 100, and thou, O valiant guesser,
must divine it within a scant measure of attempts.

## Features

- Three degrees of peril: Easy, Medium, and Hard
- Thy finest feats (fewest guesses) for each peril are etched eternally in `score.json`.
- Mark the fleeting seconds from first conjecture to crowning glory.
- After conquest or calamity, choose to spin fortune's wheel anew.

## Installation

Ensure Python 3.11 or higher graces thy realm.
```
git clone https://github.com/kr4us3r/guess-cli.git
cd guess-cli
pip install .
```

## Usage

1. `guess-cli` to commence the spectacle.
2. Elect thy peril: 1 for Easy, 2 for Medium, 3 for Hard.
3. Hurl thy conjectures (1-100); the oracle shall whisper if 'tis "higher than the stars" or "lower than the roots".
4. Triumph? Bask in laurels and temporal measure. Falter? The secret is unveiled in sorrowful mercy.
